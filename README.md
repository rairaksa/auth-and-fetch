# auth-and-fetch

## Setup
Auth and Fetch menggunakan Laravel sebagai backend.

### Installasi Auth Apps

1. `cd auth_apps`
2. `composer install`
3. `php artisan migration`
4. `php artisan serve --port=8000`

### Installasi Fetch Apps

1. `cd fetch_apps`
2. `composer install`
3. `php artisan serve --port=9000`
